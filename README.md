# 🏏 IPL Data Analysis using SQL

## 📌 Project Overview

This project analyzes IPL (Indian Premier League) dataset using SQL to extract meaningful insights related to team performance, player statistics, and match trends.

The analysis focuses on solving real-world analytical questions using structured SQL queries.

---

## 🎯 Objectives

* Analyze team performance across seasons
* Evaluate player performance metrics
* Identify match trends and patterns
* Solve business-oriented analytical questions using SQL

---

## 🗂️ Dataset Description

The dataset includes:

* Match-level data (season, teams, winner, city, toss results)
* Player-level data (batsman, bowler, runs, dismissals)
* Ball-by-ball delivery data

👉 The analysis is based on multiple real-world queries such as team wins, strike rates, and player performance 

---

## 🛠️ Tools & Technologies

* SQL (Oracle SQL Developer)
* Relational Database Concepts

---

## 🧠 SQL Concepts Used

* JOIN (multi-table analysis)
* GROUP BY & Aggregations
* Subqueries
* Filtering & Conditions
* Aggregate Functions (SUM, AVG, COUNT)

---

## 📊 Key Analysis Performed

* Matches won by each team per season
* Average strike rate of batsmen
* Toss impact on match results
* Top players based on performance
* Player dismissals against specific bowlers
* Boundary percentage analysis

---

## 🔍 Key Insights

* Certain teams consistently perform better across seasons
* High strike rate players significantly impact match outcomes
* Toss decisions influence match results
* Boundaries (4s & 6s) play a major role in scoring

---

## ✅ Validation Approach

* Maintained a structured SQL file containing all queries
* Stored query outputs for cross-verification
* Ensured accuracy and consistency of results
